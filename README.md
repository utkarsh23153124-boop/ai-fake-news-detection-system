# 📰 AI Fake News Detection System

An AI-powered fake news detection system that classifies news articles as **Real** or **Fake** using **Natural Language Processing (NLP)** and a **Bidirectional LSTM deep learning model**.

---

## 🚀 Features

- Detects fake vs real news articles
- Uses NLP techniques for text preprocessing
- Deep learning model using Bidirectional LSTM
- High accuracy classification
- Simple and easy-to-use interface (if UI added)

---

## 🧠 Technologies Used

- Python
- TensorFlow / Keras
- Natural Language Processing (NLP)
- Bidirectional LSTM
- NumPy, Pandas
- Scikit-learn

---

## 📊 Model Details

- Model: **Bidirectional LSTM**
- Task: Binary Text Classification (Fake / Real)
- Input: News text
- Output: Prediction (Fake or Real)

---

## ⚙️ Workflow

1. Data Collection
2. Data Cleaning and Preprocessing
3. Tokenization and Padding
4. Model Building (Bidirectional LSTM)
5. Model Training
6. Evaluation
7. Prediction

---

---

## 📈 Results

- Achieved **97.7% accuracy**
- Model performs well on unseen data
